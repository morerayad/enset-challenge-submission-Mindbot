# DemoCrew AI – enset-challenge-submission-Mindbot

DemoCrew AI est une application d'intelligence artificielle qui coordonne l'exécution des tâches via des agents intelligents (AI Agents). Elle automatise la gestion des tâches, améliore la productivité et facilite la collaboration.

## Fonctionnalités
- Analyse le niveau de l'étudiant
- Crée un plan d'étude personnalisé
- Collaboration entre agents pour résoudre les problèmes
- Interface simple et intuitive

## Installation
1. Cloner le projet :
   git clone https://github.com/morerayad/enset-challenge-submission-Mindbot.git
2. Installer les dépendances : 
   pip install -r requirements.txt
3. Lancer l'application : 
   python main.py

## Fichiers principaux
- `main.py` : point d'entrée de l'application
- `app.py` : logique principale des agents
- `test_crewai.py` : tests de l'application
